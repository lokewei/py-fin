# 可转债数据维度分析

## 当前数据覆盖情况

### ✅ 已有的关键维度

| 维度 | 字段 | 用途 |
|------|------|------|
| **价格信息** | 现价、转股价值、转股溢价率 | 判断估值水平 |
| **时间维度** | 剩余年限、到期时间 | 判断临期、存续期 |
| **规模信息** | 剩余规模(亿元) | 判断大/中/小市值 |
| **质地评估** | 评级、正股PB、纯债价值 | 判断安全性 |
| **收益指标** | 到期税前收益、回售收益 | 判断保本收益 |
| **触发条件** | 强赎触发价、下修触发价、回售触发价 | 判断触发状态 |
| **正股信息** | 正股价、正股代码、正股名称 | 计算触发比例 |
| **流动性** | 成交额、换手率 | 判断交易活跃度 |
| **持仓情况** | 基金持仓、转债流通市值占比 | 判断机构关注度 |

### ⚠️ 缺少的维度

| 维度 | 说明 | 影响的筛选策略 |
|------|------|----------------|
| **大股东持仓** | 5%以上股东持债情况 | 无法判断"大股东未减持"、"次新债解禁" |
| **上市时间** | 转债上市日期 | 无法精确判断"次新债"（可用到期时间-6年粗略估算） |
| **历史价格** | 多日价格数据 | 无法判断"连续N天满足条件" |
| **下修历史** | 历史下修次数和时间 | 无法判断下修意愿（但转股价带*号可部分反映） |

### 📊 可计算的衍生指标

基于现有数据可以计算：

1. **期权价值** = 现价 - 纯债价值
2. **距离保本** = 现价 - 纯债价值
3. **正股价/强赎触发价比例** - 判断是否超越强赎线
4. **正股价/下修触发价比例** - 判断是否接近下修线
5. **双低指标** = 转股价值 + 转股溢价率（数据中已有）
6. **市值分类** - 基于剩余规模分为大/中/小市值

## 筛选策略实现度

### 回避策略

| 策略 | 实现度 | 说明 |
|------|--------|------|
| 临期负收益 | ✅ 100% | 有剩余年限和到期收益 |
| 强赎线上大市值高溢价 | ✅ 100% | 所有数据都有 |
| 临期双高小市值 | ✅ 100% | 所有数据都有 |
| 次新债高估 | ⚠️ 70% | 缺少精确上市时间和解禁信息 |
| 质地不佳 | ✅ 100% | 有评级、PB、纯债价值 |

### 关注策略

| 策略 | 实现度 | 说明 |
|------|--------|------|
| 小市值高溢价近保本 | ✅ 100% | 所有数据都有 |
| 低期权临期债 | ✅ 100% | 可计算期权价值 |
| 大股东未减持 | ❌ 0% | 完全缺少此数据 |
| 两年期转债 | ✅ 100% | 有剩余年限 |

## 建议

### 短期方案（基于现有数据）

使用 `cb_screener.py` 可以实现大部分筛选策略（约85%覆盖率）。

### 长期方案（数据增强）

1. **增加大股东持仓数据**
   - 从集思录或其他数据源获取5%以上股东持债情况
   - 这是最重要的缺失维度

2. **增加历史价格数据**
   - 保存每日数据，建立时间序列
   - 可以判断"连续N天满足条件"

3. **增加上市时间字段**
   - 精确判断次新债
   - 计算解禁时间
